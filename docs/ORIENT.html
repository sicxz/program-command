What this project is

Program Command is the EWU Design program's scheduling, workload, and capacity-planning surface. Vanilla JS frontend, Supabase (Postgres) for persistence, Chart.js for visuals, no build step. Production is a static GitHub Pages site reading a public Supabase project; edits are gated by a server-side editor allowlist.
