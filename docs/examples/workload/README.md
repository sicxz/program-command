# Workload Example Files

Put example workload documents here so they are easy to find and use as export targets/templates.

Recommended filenames:
- `chair-workload-example-AY2026-27.xlsx`
- `chair-workload-example-AY2026-27.xlsm`
- `workload-template-v1.xlsx`

Use this folder for:
- Example spreadsheets from the chair
- Template spreadsheets we should match for export formatting
- Annotated reference exports

Do not use this folder for generated outputs.

Generated exports should go in:
- `output/spreadsheet/`

If the file has sensitive data, note that in the filename or keep a redacted copy here.
